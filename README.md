# 🌐 Nandini Kushwah Portfolio

A modern, responsive, and production-ready personal portfolio website built to showcase my experience, projects, technical skills, and internship journey in Software Engineering, Backend Development, Machine Learning, and AI.

📄 **Resume:** Download directly from the website  
💼 **LinkedIn:** https://linkedin.com/in/nandini-kushwah  
💻 **GitHub:** https://github.com/Nandini-kush

---

## 🚀 Features

- ✨ Modern and responsive design
- 🌙 Light and Dark mode toggle with theme persistence
- 📄 Direct resume download
- 🧠 Project showcase with GitHub and live demo links
- 💼 Experience and education sections
- 🛠 Categorized technical skills
- 📬 Functional contact form with email notifications
- 🔍 SEO optimized metadata
- ⚡ Smooth animations and transitions
- ♿ Accessible and mobile-friendly UI

---

## 🛠 Tech Stack

### Frontend
- React.js
- TypeScript
- Tailwind CSS
- Vite

### UI & Icons
- Lucide React
- Framer Motion

### SEO
- React Helmet Async

### Notifications
- Sonner

### Contact Form
- EmailJS (or Formspree/Web3Forms)

### Deployment
- Vercel / Netlify
